# Agent97 Version
 **Version**        : 0.0.3 <br>

### Getting Start with Agent97.
```bash
# Download Agent97
curl -sL https://github.com/HarryTheDevOpsGuy/agent97/raw/master/x86_64/agent97 -o /usr/bin/agent97
chmod +x /usr/bin/agent97

# To check version
agent97 -v

# To view help page
agent97 -h
```
